Enterprise WordPress Server
Project Overview

This project demonstrates the deployment and administration of a complete WordPress web server on Ubuntu Server.

The server was built from scratch inside a UTM virtual machine running on macOS.

Technologies Used
Ubuntu Server
OpenSSH
Nginx
MariaDB
PHP 8.5
PHP-FPM
WordPress
UFW Firewall
Git
GitHub
Project Goals
Deploy an Ubuntu Server
Configure SSH remote administration
Install and configure Nginx
Install and secure MariaDB
Configure PHP-FPM
Deploy WordPress
Configure Nginx virtual host
Configure firewall rules
Verify full WordPress functionality
Document the entire deployment process
